Functions

hash

cryptography.hash( options, successCallback, errorCallback )

Discussion

Computes the hash of a string

Arguments

  • _options_required

Object with key-value. Supported keys:

* data

A String value. The key must contains string to compute the hash.

* hash

A String value. The key must contains hash algorithm.

Supported algorithms:

* md5 * sha1 * sha-256 * sha-384 * sha-512

  • succesCallbackoptional

Success callback. Called when function finished without errors

Callback parameter is object which contains keys:

* hash - Contains computed the hash. (String)

  • errorCallbackoptional

Error Callback. Called when function return error.

Return Value

  • RBCPromise object

Sample

cryptography.hash( {data:"string", hash:"md5"}, function(info) { alert("md5('string') = "+info.hash); }, function (error) { alert("code: " \+ error.code + "\nmessage: " \+ error.message); });
