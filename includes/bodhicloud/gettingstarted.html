Bodhi Cloud API

The Bodhi Cloud API is a set of secure APIs that are used to store your data. To access the Bodhi Cloud API, you need to login with you user and password which is provided by Hot Schedules. The API docs are available at this location: https://api.bodhi.space/apidocs/index.html. If you enter your credentials (username and password) and click Explore, you will see a list of API which you can access for your namespace. The Bodhi API are REST APIs that returns JSON for all responses. Each API has examples and an opportunity to try it out. YOu can add data using our provided types or create your own type. NOTE: Should you create your own type, the new type will show up in the API Documentation for your namespace.
